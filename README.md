# CleanMyOtus
Heuristic-algorithm for decontamination of OTU tables.

More detailed tutorial coming soon!

## Dependencies
-Python3

-BLAST

## Usage

This script takes as input the mothur-formatted tab-delimited OTU table (see sampleOTUtable.txt for specific format of this), as well as a tab-delimited file that maps each sample to a corresponding blank (see sample_blank_map for specific format)
